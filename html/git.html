Команды Git

  • git config --global user.name "Ivan Ivanov" - Указать имя пользователя
  • git config --global user.email "mail@gmail.com" - Указать электронную почту
  • git config --list - Посмотреть настройки
  • git init - Создать репозиторий
  • git clone - Клонировать удалённый репозиторий
  • git remote add origin [ссылка на удалённый репозиторий] - Связать удалённый и локальный репозитории
  • git pull - Подтянуть изменения
  • git status - Посмотреть статус файлов
  • git add - Добавить файлы в индекс
  • git commit -m "Комментарий к коммиту" — Сделать коммит (фиксирует изменения)
  • git push - Запушить изменения. Отправляет все зафиксированные изменения с локального репозитория в удалённый. Это одна из самых важных команд, ведь все вышеописанные действия производятся в локальной копии репозитория
